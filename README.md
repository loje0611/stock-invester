# 코스닥 단타 자동매매 프로그램

키움증권 차세대 REST API 기반의 코스닥 단타 자동매매 시스템입니다.  
HTS 설치 없이 HTTP 통신만으로 동작하며, `tkinter` GUI 대시보드를 제공합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **설정창** | 시작/종료 시간, 매매 주기를 입력하는 tkinter GUI |
| **실시간 대시보드** | 보유 종목·수익률·타이머·거래 로그를 1초 간격 갱신 |
| **1분 단위 스크리닝** | 거래대금 상위 100종목에서 단가·수급·추세·가격위치 조건 필터링 |
| **모멘텀 점수 누적** | 매매 주기 내 조건 충족 시 1점씩 누적, 최고 점수 종목 매수 |
| **슬리피지 최소화** | 주기 종료 15초 전 선매도 → 정각 매수 시차 적용 |
| **블랙리스트** | 직전 턴 매도 종목 재매수 자동 차단 |

## 환경 요구사항

- **Python** 3.10 이상 (64비트)
- **Windows** (tkinter 내장)
- **외부 라이브러리**: `requests`만 필요

## 설치 및 실행

```bash
# 1. 의존성 설치
pip install -r requirements.txt

# 2. 설정 파일 생성 — 템플릿을 복사한 뒤 본인의 키움 API 정보 입력
cp config.example.py config.py
# config.py에서 아래 항목을 수정하세요:
#   - REAL_APP_KEY / REAL_SECRET_KEY / REAL_ACCOUNT_NO (실전투자용)
#   - MOCK_APP_KEY / MOCK_SECRET_KEY / MOCK_ACCOUNT_NO (모의투자용)
#   - IS_DRY_RUN = True (모의) / False (실전) 전환

# 3. 실행
python main.py
```

## 매매 흐름

```
[설정창 입력] → [토큰 발급] → [시작 시간 대기]
     ↓
┌─ 매매 사이클 (N분 반복) ──────────────────┐
│  0초~    : 1분마다 시세 조회 + 스코어링   │
│  N분-15초: 보유 종목 전량 시장가 매도      │
│  N분-14초: 누적 점수 최고 종목 확정        │
│  N분 정각: 확정 종목 시장가 매수            │
└───────────────────────────────────────────┘
     ↓
[종료 시간 도달] → [전량 매도] → [프로그램 종료]
```
